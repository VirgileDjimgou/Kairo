# AI Continuity Docs

This folder keeps the project state portable across IDEs and coding agents.

## Files

- `PROJECT_STATE.md`: verified implementation snapshot
- `NEXT_SPRINT.md`: current sprint target and acceptance criteria
- `PROMPT_UNIVERSAL_NEXT_SPRINT.md`: copy-paste prompt for the next implementation sprint
- `PROMPT_SESSION_HANDOFF_TEMPLATE.md`: reusable handoff prompt template for partial progress

## Usage

At the start of a new session, read:

1. `AGENTS.md`
2. `docs/ai/PROJECT_STATE.md`
3. `docs/ai/NEXT_SPRINT.md`

Then either:

- paste `docs/ai/PROMPT_UNIVERSAL_NEXT_SPRINT.md` into Codex, Cursor, or Copilot, or
- use `docs/ai/PROMPT_SESSION_HANDOFF_TEMPLATE.md` if the previous session ended mid-task.
