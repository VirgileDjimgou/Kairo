# OrgMind AI Prompt Pack

This pack contains Markdown configuration and guidance files for building **OrgMind AI** with Codex Desktop or GitHub Copilot.

OrgMind AI is a local-first, multi-tenant RAG platform for associations, small businesses, clubs, NGOs, and internal teams.

## Recommended usage

Copy these files into the root of your future repository and ask Codex/Copilot to treat them as the source of truth.

Start with:

1. `00_MASTER_PROMPT_CODEX.md`
2. `01_PROJECT_CONSTITUTION.md`
3. `02_ARCHITECTURE.md`
4. `03_ROADMAP_SPRINTS.md`
5. `06_PROMPTING_GUIDE.md`
