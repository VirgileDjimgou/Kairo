# Product Positioning

## 1. Product Name

Working name: **OrgMind AI**

Alternative names:

- TenantWise RAG
- CivicRAG Platform
- AssociAssist AI
- Secure Org Assistant
- Community Knowledge AI

---

## 2. One-liner

OrgMind AI is a local-first, multi-tenant RAG platform that lets associations and small organizations turn their internal documents and records into a secure AI assistant.

---

## 3. Portfolio Pitch

```text
A secure, local-first AI knowledge assistant for organizations, built with FastAPI, Vue 3, PostgreSQL, Qdrant, MinIO, Ollama, Docker and Cloudflare Tunnel. It supports multi-tenancy, document ingestion, access-controlled retrieval, source citations, admin dashboards and structured association/business modules.
```

---

## 4. Target Market

Initial target users:

- associations
- sports clubs
- cultural groups
- NGOs
- community groups
- small businesses
- internal company departments

---

## 5. Core Value Proposition

OrgMind AI helps organizations:

- centralize scattered knowledge
- reduce repeated questions
- improve transparency
- protect private data
- make rules and decisions searchable
- manage contributions and records
- run without expensive cloud infrastructure

---

## 6. Differentiators

- local-first deployment
- Cloudflare Tunnel exposure
- tenant-aware RAG
- citations-first AI responses
- structured business modules
- flexible provider architecture
- portfolio-grade open architecture

---

## 7. Demo Story

A demo should show:

1. Admin creates an organization.
2. Admin uploads statutes, rules, payment records, and meeting minutes.
3. The system ingests documents.
4. Member asks a question about rules.
5. Bot answers with citations.
6. Member checks personal contribution balance.
7. Admin sees ingestion and audit logs.
8. Unauthorized user cannot access private data.
9. App is running locally but available through Cloudflare Tunnel.

---

## 8. Commercialization Path

Stage 1:

- GitHub portfolio project
